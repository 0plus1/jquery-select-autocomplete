jQuery Select Autocomplete Demo

Normally, you'd keep the select list hidden (which this plugin does automatically), but for the purposes of this demo we'll keep it visible so you can see the selected item change when you make a selection via the auto-complete text input.

The select list has the items 'One' through 'Ten' written out as words. Try typing in the input box and see what happens when you pick a value from the auto-complete list.

Also, the text input box's value will default to the display text of the option with the selected="selected" attribute in the select list, if one is present.
